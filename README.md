# Ejercicio de Entornos - DAM

Proyecto Java básico con Maven.

## Descripción
Este proyecto contiene un simple programa "Hello World" en Java.

## Estructura
```
src/main/java/com/ejemplo/
└── HelloWorld.java
```

## Cómo compilar y ejecutar

```bash
javac -d bin src/main/java/com/ejemplo/HelloWorld.java
java -cp bin com.ejemplo.HelloWorld
```

## Output esperado
```
¡Hola mundo desde Java!
```

## Requisitos
- Java 11 o superior
- Maven (opcional)
